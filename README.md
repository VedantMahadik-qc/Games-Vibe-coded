# Space Shooter

A fast-paced arcade **space shooter** built with Python and Pygame.  
Features multiple enemy types, boss fights, powerups, particles, and a persistent high score.

---

## Game Modes

- **Desktop version (with audio)**  
  File: `space_shooter_audio.py`

- **Browser version (no audio)**  
  Built separately using `pygbag` for web deployment.

---

## Requirements

- Python 3.10+  
- `pygame` library

Install pygame with:

```bash
pip install pygame
